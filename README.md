# vz.services

Roblox Studio scripting services site with an order form and the vzorkin assistant.

## Local

```bash
cp .env.example .env
npm start
```

## Vercel Environment Variables

Set these in Vercel Project Settings:

Add `MISTRAL_API_KEY`, `MISTRAL_MODEL`, and `DISCORD_WEBHOOK_URL`.

Keep the repository private so API keys and deployment history stay protected.
